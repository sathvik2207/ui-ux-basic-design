# Usability Issues Identified

## Issue 1

### Description

Some users initially overlooked the Search Bar.

Severity:

**2 — Minor**

Recommendation:

Increase visual prominence through spacing or icon emphasis.

---

## Issue 2

### Description

Several users expected the Floating Action Button to remain visible while scrolling.

Severity:

**2 — Minor**

Recommendation:

Keep the Floating Action Button persistent on long pages.

---

## Issue 3

### Description

A few users wanted more task categories.

Severity:

**2 — Minor**

Recommendation:

Allow custom categories.

---

## Issue 4

### Description

Some participants searched for notification settings inside the profile instead of the settings screen.

Severity:

**3 — Moderate**

Recommendation:

Add a direct shortcut to notification settings from the profile page.

---

## Issue 5

### Description

Two participants wanted additional sorting options.

Severity:

**3 — Moderate**

Recommendation:

Provide filters such as:

* Due Date
* Priority
* Category
* Completion Status

---

## Issue 6

### Description

Some users requested recurring tasks.

Severity:

**3 — Moderate**

Recommendation:

Support recurring daily, weekly, and monthly tasks.

# Accessibility Observations

Positive findings:

* Text was easy to read.
* Buttons were large enough to tap comfortably.
* Color contrast was sufficient.
* Navigation labels were understandable.
* Form fields were clearly labeled.

Suggested improvements:

* Add support for larger text sizes.
* Improve focus indicators for keyboard navigation.
* Increase spacing around small interactive elements.

# Final Evaluation

The usability testing indicates that TaskFlow provides an intuitive and efficient experience for its target users. Participants were able to complete essential tasks with minimal guidance and expressed high satisfaction with the application's navigation, visual design, and overall simplicity.

The identified usability issues were minor to moderate and can be addressed in future design iterations without requiring significant changes to the application's structure.

---

# Conclusion

The usability testing successfully validated the core design decisions made throughout the TaskFlow project. Users found the application easy to learn, efficient to navigate, and enjoyable to use. The feedback collected during testing will guide future improvements, ensuring that TaskFlow continues to evolve into a more accessible, feature-rich, and user-centered productivity application.
