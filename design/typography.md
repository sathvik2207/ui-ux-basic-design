# Typography

## Project Title

**TaskFlow – A Smart Task Management Mobile Application**

---

# Introduction

Typography plays a critical role in creating a clear, readable, and visually appealing user interface. The TaskFlow typography system is designed to establish a consistent visual hierarchy, improve readability, and support accessibility across all screens.

The selected fonts emphasize simplicity and professionalism while ensuring excellent readability on mobile devices.

---

# Typography Goals

The typography system aims to:

* Improve readability on small screens.
* Establish a clear content hierarchy.
* Maintain consistency throughout the application.
* Reduce cognitive load by using predictable text styles.
* Support accessibility with appropriate font sizes and spacing.
* Create a modern and approachable visual identity.

---

# Font Families

## Primary Font

**Poppins**

### Usage

* Application title
* Screen headings
* Section titles
* Buttons
* Navigation labels

### Why Poppins?

Poppins is a geometric sans-serif typeface with a clean and modern appearance. Its balanced letterforms make headings visually strong while remaining easy to read.

---

## Secondary Font

**Inter**

### Usage

* Body text
* Descriptions
* Form labels
* Helper text
* Notifications
* Error messages

### Why Inter?

Inter is optimized for digital interfaces and provides excellent readability at smaller sizes. It performs well across different devices and screen resolutions.

# Text Alignment

The application uses alignment based on content type:

| Element           | Alignment |
| ----------------- | --------- |
| Page Titles       | Left      |
| Body Text         | Left      |
| Form Labels       | Left      |
| Buttons           | Center    |
| Navigation Labels | Center    |
| Statistics        | Center    |
| Dialog Content    | Left      |

Left alignment improves readability and scanning, while centered alignment is reserved for short interface elements.

# Readability Guidelines

To improve readability:

* Avoid long paragraphs.
* Use concise labels and messages.
* Maintain consistent spacing between headings and body text.
* Use sentence case for interface text.
* Avoid excessive capitalization.
* Keep line lengths appropriate for mobile screens.

---

# Accessibility Guidelines

The typography system supports accessibility by:

* Using a minimum body text size of 16 px.
* Maintaining sufficient contrast between text and backgrounds.
* Avoiding decorative fonts for functional content.
* Ensuring text remains readable when scaled by users.
* Providing clear distinctions between headings and body text.
* Supporting screen readers through semantic text hierarchy.

---

# Responsive Typography

On smaller screens:

* Preserve the established hierarchy.
* Prevent text clipping by allowing wrapping where appropriate.
* Maintain comfortable line spacing.
* Avoid reducing body text below 16 px.

---

# Typography Do's

* Use consistent font sizes.
* Maintain clear heading hierarchy.
* Keep text concise.
* Prioritize readability over decoration.
* Use bold text sparingly to emphasize important information.

---

# Typography Don'ts

* Do not mix multiple font families unnecessarily.
* Do not use all-uppercase for long text.
* Do not rely solely on color to indicate importance.
* Do not overcrowd text with insufficient spacing.
* Do not reduce body text below accessible sizes.
